Caesar Cipher

A Caesar cipher is a basic encryption method in which each letter of a given message is substituted with another letter further along in the alphabet.

These substitutions are done in systematic shifts in which the alphabet is permutated by one letter for each rotation. For example a ROT1 shift would suggest that all A's used in the message are replaced with B's, and consequently all B's are replaced with C's, and so on.
